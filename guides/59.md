# Operator Troubleshooting Checklist: Vacuum Lift System (Glue Line )

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Suction Loss / Parts Dropping | - Material not being grabbed<br>- Heavier pieces dropping<br>- Panels falling off | - Check for torn or worn suction cups<br>- Check for torn air hoses<br>- Inspect panel for excess glue (can make surface slippery) | - If suction cups or hoses are torn and need replacement<br>- If vacuum gauge is not reaching/holding 30" Hg |
| Gasket Wear | - Rubber gaskets worn | - Inspect bottom end gaskets for wear | - If gaskets require replacement |
| Handle Height Issues | - Operators reaching too far<br>- Difficulty grabbing handle | - Verify if handle height is appropriate for operator | - If handle needs to be lowered/adjusted per Novatech instructions |
