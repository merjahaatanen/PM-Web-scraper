# Operator Troubleshooting Checklist: 2 gallon glue tank with hand held glue nozzle gun

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Glue Flow Blockage | - Glue nozzle not working<br>- Glue chunks coming out | - Check if glue drum is empty or contains solidified chunks<br>- Remove glue container<br>- Fill container halfway with hot water<br>- Increase PSI to push debris/chunks out of air line and nozzle | - If blockage persists after flushing with hot water |
| Glue Gun Mechanical Failure | - Trigger not retracting<br>- Glue continues to flow after releasing trigger<br>- Must manually pull trigger back to stop flow | - None (Operator cannot repair internal gun mechanism) | - Immediately when trigger fails to spring back |

## Work Orders & Parts Reference

| Issue | Related Work Order #s | Parts Often Needed |
| :--- | :--- | :--- |

| Glue Gun Mechanical Failure | 6255 | Glue Gun |
