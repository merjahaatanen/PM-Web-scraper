# Operator Troubleshooting Checklist: Evolve Double Head Drilling Machine

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Alignment & Squaring | - Back bar/fence not aligned<br>- Corner stops loose<br>- Drilling stops not squared<br>- Guide is loose | - Check if hardware/locknuts on fences are loose<br>- Check if stops have shifted from being flush<br>- Use a square ruler to verify fence alignment | - If hardware cannot be tightened<br>- If phenolic spacers/permanent blocks are needed<br>- If machine requires electrical rewiring |
| Clamping & Material Hold | - Left side clamp not working/hitting frame<br>- Right side material clamp stuck<br>- Clamp only turns halfway<br>- Vacuum mount falling off | - Check if the clamp has shifted position<br>- Check if the rubber bolt (finger) is too low and hitting the frame<br>- Clean all sensors<br>- Test with scrap material to verify hold | - If clamp continues to hit the metal frame after adjustment<br>- If air supply fittings need removal to clear back pressure |
| Sensor & Activation | - Sensor not turning on<br>- Drilling not activating<br>- Machine not working properly | - Check if sensors are loose or wiggling<br>- Ensure sensor is flush and hasn't moved out of position<br>- Adjust sensor sensitivity | - If locknuts are stripped<br>- If sensor requires replacement |
| Drill Performance | - Part holder getting stuck midway<br>- Pin is too tight going into the hole | - Check part holder alignment<br>- Use a hole gauge to check pin fit | - If pin is tight (indicates drill needs replacement) |
| Pneumatics | - Audible air leak on back side | - Identify location of leak | - If air hose needs replacement |
