# Operator Troubleshooting Checklist: Gannomat Index 330 Trend/PRO (Solid)

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| Barcode Reader & Program Loading Issues | - Barcode reader not working<br>- Barcode reader not scanning<br>- Machine not loading drilling programs<br>- Cannot access Masterfile<br>- Teamviewer inaccessible<br>- System lagging to bring up program | - Visually check barcode reader and its USB cable for obvious damage or disconnection.<br>- If instructed and safe, try clearing old programs or the trash bin from the machine's computer to improve system speed. | - If barcode reader is damaged or cable is faulty.<br>- If programs still won't load or Masterfile is inaccessible.<br>- If system lagging persists after clearing programs.<br>- For any network or IT-related software issues. |
| Dust Collection Problems | - Dust collection pipe raised<br>- Machine not picking up drilling debris<br>- Drilling debris on the table<br>- Dust collection cap (for drill bit replacement) is split or damaged | - Visually inspect the dust collection pipe to ensure it is at its original, marked height and position.<br>- Check the dust collection cap for cracks or damage. | - If the dust collection pipe needs adjustment or securing.<br>- If the dust collection cap is split or broken. |
| Machine Power & Computer Startup Issues | - Machine won't turn on<br>- Machine display is off<br>- Computer shows "failed to start up normally" message<br>- Monitor screen stopped working | - Ensure the machine was properly shut down at the end of the previous shift. Refer to start-up and turn-off instructions posted by the computer.<br>- Perform a controlled machine restart by following proper shutdown and startup procedures. | - If the machine does not turn on.<br>- If the display remains off.<br>- If the computer fails to start normally after a proper restart attempt.<br>- If the monitor screen is completely black or unresponsive. |
| Sensor Malfunctions | - Sensors not activating<br>- Sensors not picking up pieces (e.g., doors) | - Visually inspect the affected sensors for any obstructions (dust, debris) or visible damage.<br>- Gently clean the sensor area if dust or debris is present. | - If sensors are not activating or picking up pieces despite visual checks.<br>- If sensors appear physically damaged or misaligned. |
| Drilling Performance & Air Pressure | - Air regulator bumped or hanging off<br>- Machine taking longer to drill holes | - Visually check the air regulator to ensure it is securely mounted and not damaged.<br>- Check the air pressure gauge (if accessible and safe) to ensure it's at the correct operating pressure.<br>- If required by routine, replace the drill bit (standard operator task). | - If the air regulator is damaged, broken, or cannot be secured.<br>- If air pressure is low or cannot be adjusted by operator controls.<br>- If drilling performance issues persist after checking air and replacing the drill bit. |
| Drilling Accuracy & Mechanical Alignment | - Drilling is off by 1/16" or more<br>- Machine appears to have shifted<br>- Front table is not level | - Immediately stop production if drilling accuracy is affected.<br>- Report any noticeable shift in drilling or unlevel surfaces. | - Immediately for any drilling accuracy problems.<br>- Immediately if the machine's position has shifted or the table is unlevel. |
| Specific Mechanical Failures | - Clamp cap popped off, ring fell out<br>- Drill bit not holding in place by set screw | - Visually inspect clamps for missing parts or damage.<br>- Check the drill's set screw area for any loose or broken components. | - For any broken or malfunctioning clamps.<br>- If the drill bit cannot be securely held in place. |
| Axis Movement Error | - Error message "29: Dynamic Error Servodrive X-Axis!" appears<br>- Machine stops working<br>- Error reoccurs after reset | - Try performing a machine reset once. | - Immediately if the error persists after one reset attempt.<br>- For any recurring axis movement errors. |
| Backstop & Table Alignment | - Holes not lining up during gapless drilling<br>- Doors rubbing against side panels when moving to machine | - Visually inspect the backstop for alignment or damage.<br>- Check for any obstructions or rubbing on the table top when loading large doors (58''). | - If the backstop needs to be reworked or replaced.<br>- If side panels or table extensions need adjustment to prevent rubbing. |
| Camera Fixture Issues | - Camera fixture loose or missing | - Visually check that the camera fixture frame is securely bolted to the machine. | - If the camera fixture frame is loose or requires installation. |
