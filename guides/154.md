# Operator Troubleshooting Checklist: CNC Drilling Machine, Automatic Leveling Bar

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Drill Alignment & Accuracy | - Holes not centered<br>- Holes not straight<br>- Leveling bars skewed to one side | - Verify material is pushed firmly against back stops<br>- Check if side stop is square to back stop | - Guides need squaring<br>- Drill head needs adjustment<br>- Servo motor replacement required |
| Drill Bit & Hole Quality | - Smoke smell during operation<br>- Holes not deep enough<br>- Chipping on bottom side of material | - Inspect bit for burnout/blackening<br>- Verify drill bit size is correct for the part<br>- Replace dull or damaged drill bits | - Drill descent speed needs adjustment<br>- Motor hold-down bolts are loose |
| Dust & Vacuum Issues | - Dust buildup on stiles after drilling<br>- Vacuum not suctioning | - Check vacuum filter for clogs<br>- Inspect vacuum line for disconnects<br>- Check if suction hose is caved in or blocked | - Internal vacuum branches are clogged<br>- Vacuum sensors require cleaning |
| Machine Setup & Sensors | - Green light is ON despite sensors not being triggered<br>- Machine won't drill 24" stiles | - Ensure folding extension top is extended<br>- Use adjustable support to keep material flat and square | - Sensor sensitivity needs adjustment<br>- Extension support bracket is broken/missing |
| Tooling Wear | - Hard to tighten or remove drills<br>- Chuck feels worn | - Inspect chuck for debris or damage | - Chuck requires replacement |
