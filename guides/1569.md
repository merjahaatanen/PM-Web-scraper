# Operator Troubleshooting Checklist: CNC Drill Setup Gage-1080

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Drill Depth Issues | - One side drilling deeper than other<br>- Uneven hole depth | - Check material thickness (ensure it is at 7.5)<br>- Inspect drill points for uneven edges or height differences | - If drill point is uneven<br>- If set screws are loose or need replacement |
| Control Response | - CNC not responding<br>- Drill will not activate | - Ensure machine was turned off correctly<br>- Perform a system reset<br>- Verify machine is "homed" | - If machine remains unresponsive after reset |
| Pneumatic Leaks | - Air hose leaking | - Identify location of leak | - To replace air hose fittings |
