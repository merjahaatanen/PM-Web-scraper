# Operator Troubleshooting Checklist: Chop Saw (corner guard pack out Lam.)

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Dust Collection | - Dust going everywhere<br>- Too much dust during cutting<br>- Vacuum clogged | - Check vacuum hose for breaks or holes<br>- Empty booth vacuums<br>- Clean vacuum filters | - If hose needs replacement<br>- If suction remains weak after cleaning |
| Power & Operation | - Chop saw stopped working | - Check and reconnect the power connector | - If machine still won't power on<br>- If tension spring is broken |
| Blade Maintenance | - Blade needs replacement | - None (Operator should not replace) | - When new blades arrive or blade is dull |
