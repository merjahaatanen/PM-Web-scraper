# Operator Troubleshooting Checklist: Forklift # T20

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Oil Leaks | - Oil puddles on floor overnight<br>- Wet hoses under lift | - Inspect floor for puddles<br>- Check hoses for wetness/drips | - Leak is between motor and gearbox<br>- Hoses are wet or leaking |
| Electrical & Controls | - Forklift stops functioning (acts like seat belt is unbuckled)<br>- Level function button not working<br>- Charging station cable loose | - Check seat belt connection<br>- Test level button again<br>- Check charger cable connection | - Machine won't start/function<br>- Button is faulty<br>- Charger cable is loose or damaged |
| Operator Station | - Seat belt not working<br>- Seat cushion torn or taped | - Check seat belt latch<br>- Inspect seat fabric | - Seat belt needs replacement<br>- Seat cushion is in bad shape |
| Battery & Charging | - Battery acid leaking on floor | - Use towels to clean area<br>- Place bucket under leak to catch acid | - Battery is cracked<br>- Acid is leaking |
| Brakes | - Worn out brake performance | - Test braking distance | - Brake assembly is worn out |
