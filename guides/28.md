| :--- | :--- | :--- | :--- |
| Drive System | - Lift stopped driving<br>- Machine will not move | - Check if seatbelt is properly buckled<br>- Ensure seatbelt switch is engaged | - If machine still will not move after checking seatbelt |
| Tire Condition | - Rear steer tires are worn out | - Visually inspect tire tread for wear | - When tires are worn and require replacement |
| Pedal Assembly | - Broken pedal | - Check for physical damage to the pedal | - If the pedal is broken or non-functional |
