# Operator Troubleshooting Checklist: Evolve cell Pop UP table

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| Pop-up Table Not Moving | - Scissor lift not going up or down. | - Check if the lift table is securely plugged into the wall outlet.<br>- Check the wall receptacle where the lift table is plugged in for a tripped reset button (e.g., GFCI). Press it to reset if tripped. | - If the lift table is plugged in but still not moving after checking the outlet.<br>- If no tripped reset button is visible or if resetting it doesn't resolve the issue.<br>- If the issue recurs after attempting troubleshooting steps. |
| Parts Falling Between Rollers | - Parts fall between the rollers when flipping them on the table. | - Observe if parts are consistently falling into gaps or between rollers during operation. | - If parts are consistently falling into gaps on the table, indicating a need for physical guards or adjustments. |
