# Operator Troubleshooting Checklist: 3/4 Thomas return system

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Belt Failure | - Belt snapped off<br>- System stopped working | - Visually inspect belt for breaks or slipping | - If belt is snapped or needs replacement |
| System Misalignment | - Return system not in original location<br>- Corner rail loose<br>- Pop-up table nuts loose | - Check if system has shifted position<br>- Inspect corner rail and pop-up table nuts for looseness | - If system needs to be moved back into place<br>- If anchoring or tightening of hardware is required |
| Mechanical Noise | - Loud screeching noise | - Listen for noise coming from the pulley area | - If belt pulley or shaft has shifted out of position |
| Sensor/Bracket Failure | - System not working properly<br>- Arm will not stay up | - Check the capacitive prox sensor arm for stability | - If sensor bracket or weld is broken |
