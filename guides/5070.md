# Operator Troubleshooting Checklist: Edge finisher Pop UP table

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Pop Up Table Function | - Table stopped working properly<br>- Table came down abruptly | - Check if the table is obstructed<br>- Verify air supply to the unit | - If table does not rise/lower correctly<br>- If a cylinder needs replacement |
| Roller Table Adjustment | - Roller table before glue machine needs assistance | - Manually adjust the roller position to align with workflow | - If adjustment does not resolve the issue |
| Hardware & Support | - Kick plate hardware fell off<br>- Support padding for stiles is in bad shape | - Inspect kick plate for loose screws<br>- Inspect padding for wear or tears | - If screws need drilling/tapping<br>- If padding needs full replacement |
