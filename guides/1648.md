# Operator Troubleshooting Checklist: Laminate Cell

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Air Table Issues | - Air does not turn on automatically<br>- Air not blowing before glue rollers<br>- Edging chipping on components | - Check if the activation rod/flap is properly positioned<br>- Verify there are no unauthorized screws added to the gate | - Actuator board is loose or adhesive failed<br>- Air hose is broken<br>- Activation rod requires tape/tightening |
| Glue System Issues | - Clogged glue line/nozzle<br>- Glue gun not working<br>- Rollers not rotating | - Check nozzle for dried glue buildup | - Glue spreader chain is broken<br>- Master link or sprocket needs replacement<br>- Internal glue gun failure |
| Table & Alignment Issues | - Screws popping up from CNC table<br>- Pop-up table not working<br>- Parts hitting glue spreader (table too high) | - Clear debris from under table surfaces<br>- Inspect for missing or loose screws | - Mounting brackets are broken<br>- Table legs require height adjustment<br>- Mechanical failure of pop-up mechanism |
| Power Issues | - Hand router outlet has no power | - Check if other nearby outlets are also dead | - Breaker (L428) has tripped |
