# Operator Troubleshooting Checklist: Return system,Thomas,Evolve cell

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Belt/Band Failure | - Orange band come off<br>- Belt snapped<br>- Bands splitting or ripping<br>- Bands not spinning | - Check if orange band has simply come off and can be reset<br>- Inspect bands for ruptures or splitting | - Belt is snapped<br>- Band is ripped/split<br>- Rod turning the arms is broken |
| Gearbox & Motor Issues | - Weird sounds/noises from gearbox<br>- Oil leaking on floor<br>- Gearbox making noise during use | - Check for oil leaks/wet floor under the machine | - Any loud or unusual noise from gear box<br>- Evidence of oil leaks<br>- System running at slow speed |
| Roller & Shaft Problems | - Shaft is wobbly<br>- Rollers cracked or damaged<br>- Rods moving/loose<br>- Product not coming out straight | - Inspect rollers for cracks or scoring marks<br>- Check if roller rods are loose | - Bearings need replacement<br>- Shaft is wearing down<br>- Roller sleeves are damaged |
| Power & Control Issues | - Return system does not turn back on | - Check if machine is powered on | - Fuses or contactors need checking<br>- Cable hardware in controller box is loose |

## Work Orders & Parts Reference
| Issue | Related Work Order #s | Parts Often Needed |
| :--- | :--- | :--- |
| Belt/Band Failure | 1207, 4613, 6916, 10838, 11447, 11458 | Forward drive belt (A144 / 6L203G), Outfeed orange round belts |
| Gearbox & Motor Issues | 7911, 10047 | Gear reducer, Motor, Oil seal |
| Roller & Shaft Problems | 1266 (Specialist alignment), 1577, 6523, 7932 | End bearing, Hex shaft, Shaft stabilizer, Rollers, Roller locks |
| Power & Control Issues | 9458 | None operator-replaceable |
