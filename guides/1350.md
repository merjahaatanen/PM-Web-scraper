# Operator Troubleshooting Checklist: FLIP TABLE

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Rubber Gasket Issues | - Rubber piece in middle is loose<br>- Rubber is sticking up/causing bumps<br>- Rubber hinge/fold came out<br>- Rubber is missing or detached | - Check if rubber can be pushed back down to smooth surface<br>- Use T-5 tool to loosen gasket and pull it flat/down to remove bump<br>- Pull rubber back to original flat position | - Rubber is completely torn or missing<br>- Hardware securing the rubber is broken |
| Table Top Fasteners | - Table top is loose/undone<br>- Screws are sticking up/scratching parts<br>- Screws no longer holding top down | - Visually inspect for screws that have popped up or lost threads | - Screws are stripped or broken<br>- Hardware needs to be replaced with bolts/nuts/lock washers |
| Alignment & Stability | - Table is uneven<br>- Parts coming to a dead stop at infeed<br>- Table moves/shifts over roller table<br>- Table is not level | - Check if table is shifted and needs repositioning<br>- Verify if part is being pushed back into the top of the table | - Broken welds found on frame<br>- Table feet are missing or damaged<br>- Table requires anchoring to the floor |
| Pneumatic Issues | - Air hose is leaking air | - Check for audible hissing or visible hose damage | - Hose requires repair or replacement<br>- Air leak is internal to the cylinder |
