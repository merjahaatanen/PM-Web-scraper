# Operator Troubleshooting Checklist: 1/2 " Edge Finisher, Solid, Technolegno/ Universal 280

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| Machine Not Starting / Power / Motor Fault | - Machine will not turn ON / turn back on<br>- Thomas Return System stop working<br>- Motor fault error showing<br>- Yellow light is on<br>- Small blade cutter not activating / not responding to ON button | - Check if the power supply switch for the return system is ON.<br>- If the machine stopped after beam height adjustment, reset the motor controller.<br>- Check the electric control panel; if an overload relay is down, flip it back ON.<br>- If the machine does not start from the ON button, try resetting control relays and ensuring button connections are firm.<br>- If the machine won't turn on after a power outage, try resetting the main controller. | - If troubleshooting steps do not resolve the issue.<br>- Motor error light is on and does not clear.<br>- Machine still does not turn on after all checks. |
| Orange Belt Issues | - Orange belt broken<br>- Orange belt getting caught on parts<br>- Orange belt is loose | - Carefully inspect the orange belt for breaks or looseness.<br>- If a belt segment is separated, attempt to reconnect or fix it if you are trained to do so safely.<br>- If the belt is loose or getting caught, visually inspect for obvious snags or misalignments. | - If the orange belt is broken and cannot be reconnected safely by an operator.<br>- If the belt is persistently loose or getting caught.<br>- If the belt needs replacement. |
| Air / Vacuum Hose / Dust Collection Issues | - Machine not starting (due to a torn hose)<br>- Air leaking from a hose<br>- Air hose to pop-up table burst<br>- Machine stopped working (due to broken air hose)<br>- Machine full of dust everywhere<br>- Vacuum hose is cracked<br>- Vacuum/dust collector disconnected | - Visually check all accessible air and vacuum hoses for tears, cracks, or disconnections.<br>- If an air hose is found disconnected, try to reconnect it if possible and safe.<br>- If a small crack is found in a vacuum hose, temporarily fix it with duct tape to minimize dust until maintenance arrives. | - If a hose is torn, burst, or broken and cannot be easily reconnected or temporarily taped.<br>- If air is consistently leaking.<br>- If the machine is collecting excessive dust despite temporary fixes. |
| Buffer Mechanism Issues | - Pin to change the buffer doesn't stay locked<br>- Buffer mechanism is not lining up<br>- Buffer is stuck, won't move when trying to pull it back | - If the buffer is stuck, try pulling the lock mechanism back firmly while attempting to turn and move the buffer. | - If the buffer pin does not stay locked.<br>- If the buffer mechanism is not lining up correctly.<br>- If the buffer remains stuck after attempting to pull the lock back. |
| Cutter/Blade Issues | - Main cutter sounds rough<br>- Bottom blade chipping<br>- Cutting blade motor makes loud noise | - Listen for unusual noises from the cutter or motor.<br>- Visually inspect the blade for obvious chipping or damage (do NOT touch the blade). | - Any unusual sounds from the cutter or motor.<br>- Bottom blade is chipping.<br>- Any issues requiring blade adjustment or replacement. |
| Panel/Part Jamming or Getting Stuck | - Panel stops midway in the machine<br>- Panel gets wedged<br>- Samples get stuck underneath the infeed guide | - IMMEDIATELY hit the E-STOP.<br>- Carefully inspect the machine for wedged or stuck panels/samples.<br>- Remove any obstructions carefully.<br>- If the machine stopped due to the jam, reset any tripped breakers. | - If panels are repeatedly getting stuck or wedged.<br>- If you cannot safely remove a stuck panel or sample.<br>- If adjustments to rollers or guides are needed. |
| Feed Track / Roller Issues | - Feed track guides are worn out<br>- Belt is making noise (from feed track)<br>- Rollers are bent, causing scratches on parts | - Visually inspect the feed track guides and rollers for visible wear, damage, or bending. | - If feed track guides appear worn.<br>- If rollers are bent or causing scratches.<br>- If the feed track belt is making unusual noise. |
| Safety Switch Not Holding | - Pinch point emergency switch is not holding | - (No operator-safe troubleshooting steps for this issue.) | - If any safety switch is not operating correctly or not holding. |
| Mechanical Damage: Leveling Leg | - One of the leveling legs on the returning table broke | - Visually inspect the leveling legs for damage. | - If a leveling leg is broken or damaged. |
