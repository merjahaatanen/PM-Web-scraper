# Operator Troubleshooting Checklist: Router, CNC, Anderson Stratos/Nest TC+D

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| Table Gasket/Seal Damaged (Vacuum Loss) | - Operator advised to replace the table gasket<br>- Table jig gasket needs to be replaced/its ripped/torn up/moving/damaged<br>- Rubbers on the suction table needs to be replaced<br>- Black seal for gapless board needs change<br>- Operator requested new gasket<br>- Gasket came apart<br>- The robber in the table is worn out<br>- Robber on the top is worn out<br>- The robber on the table ripped off<br>- Supervisor notify the robber seal on the table needs to be replaced<br>- Vacuum seal on the table is broken<br>- Robber gasket is worn out<br>- Bottom seal of the table needs to be replaced<br>- Top robber gasket on the table warned out<br>- Gapless table gasket worn out<br>- Robber seal in the table is broken<br>- New gapping table created, seal needed | - Visually inspect the table gasket/seal for any rips, tears, or signs of wear.<br>- Check if the gasket is moving or sticking up too much from the jig. | - If the gasket/seal is ripped, torn, worn out, or appears damaged.<br>- If the gasket is not staying in place or causing vacuum loss.<br>- If a new seal is required for a new gapping table or gapless board.<br>- If the table jig slot seems too shallow for the gasket. |
| Low Vacuum/Suction (Table/System Leaks) | - Machine vacuum turned off by itself (overload alert)<br>- Vacuum not working<br>- Lost suction along with the pins not holding in place<br>- No air pressure holding the components down<br>- Anderson air suction pressure too low<br>- Cutout table not holding pressure correctly<br>- Anderson vacuum system not working | - Check the cutout table for any through holes and fill them temporarily with rubber if safe to do so.<br>- Check that the alignment polls are properly seated and not leaking air. | - If display shows an overload alert for the vacuum.<br>- If vacuum is not working and no obvious gasket issue or clogged filter.<br>- If the table is damaged and causing suction loss.<br>- If alignment polls are leaking air.<br>- If the problem persists after checking for table holes. |
| Vacuum System Malfunction (Filter/Motor/Solenoid) | - Vacuum not working<br>- Trouble with the vacuum<br>- Gauge in the Andy is not working<br>- Anderson router is not picking up the dust from the parts<br>- Motor sounds like it is struggling | - Open and clean the air filter of dust.<br>- Clear out air lines. | - If vacuum is not working after cleaning filters and air lines.<br>- If the vacuum gauge is not working.<br>- If the motor sounds like it is struggling or showing an overload alert.<br>- If dust is not being picked up and air lines/filters are clear. |
| Tooling/Bit Issues | - Router was not gapping properly and was gapping too deep<br>- Tool carousel button does not work<br>- Cannot remove the tool holder to replace the router bit<br>- The bit got stuck and won't release tool<br>- Tool carousel hitting each other<br>- Tool touch off stays 8'' above the tool touch off tool | - If a bit is stuck, try clearing the AIR LINE.<br>- If the tool carousel button doesn't work, manually remove the tool by disengaging the doors to the tool carousel (if trained and safe).<br>- If gapping is too deep, check the tool offset setting in the machine controls (e.g., ensure it's at '11' if it should be). | - If bit remains stuck after clearing air line.<br>- If tool carousel button is broken or unresponsive.<br>- If unable to manually remove the tool holder safely.<br>- If tool carousel is hitting, out of sequence, or doors are not closing properly.<br>- If tool offset setting correction does not resolve gapping issue, or if tool touch-off is consistently wrong. |
| Monitor/Computer Display Issues | - Display says fan failure<br>- Monitor does not turn on<br>- Screen on the Andy won't turn on<br>- Monitor unresponsive / goes to power saving mode | - Check if the computer fan is dislodged; try to gently secure it back in place if safe and accessible.<br>- Try manually turning on the computer using the button behind the small front door on the control panel. | - If display shows "fan failure" and the fan cannot be easily secured.<br>- If monitor is unresponsive, does not turn on, or goes into power saving mode.<br>- If manual power on does not resolve the issue or the computer button on control panel is not working.<br>- If an error message regarding date/time appears. |
| Dust Collection Hose Damaged | - Main duct hose ripped in several sections<br>- Something is bouncing around inside the main duct hose<br>- The hoes on the Andy ripped | - Visually inspect the main dust collection hose for any rips, tears, or damage.<br>- Listen for unusual noises or signs of obstruction in the hose. | - If the dust collection hose is ripped, damaged, or has a slit.<br>- If you suspect an obstruction is inside the hose that cannot be safely removed. |
| Sensor Alarms | - Keyence photo sensor alarm has an error<br>- Window has contaminants<br>- Safety area sensor is sometimes triggered and stays like that | - Gently clean the Keyence photo sensor window, looking for any visible contaminants or dust.<br>- If safe, try to clear the alarm on the machine display. | - If cleaning the sensor window does not clear the alarm.<br>- If the sensor appears to be hit or physically damaged.<br>- If the safety area sensor is frequently or persistently triggered. |
| Low Oil Level / Oil Mist Alarm | - Oil is low<br>- Error popping about oil mist and low oil | - Check the oil level indicator on the machine. | - If the oil level is low.<br>- If an error message about oil mist or low oil appears. |
| T.C Alarm (Internal Battery) | - Machine throwing a code (2130) T.C Alarm<br>- T.C Alarm code (2030) | - Note the exact alarm code displayed on the machine. | - Immediately report any T.C Alarm code to maintenance for technical support. |
| Cover Holding Cylinders Malfunction | - The cylinders that holding the cover need to be replaced | - None (This is a maintenance-only repair). | - If the cylinders holding the cover are failing or need replacement. |
