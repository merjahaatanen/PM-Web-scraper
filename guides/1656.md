# Operator Troubleshooting Checklist: Drill Jig - OS Door Hinge

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Loose or Missing Hardware | - Missing screws<br>- Parts feeling loose | - Inspect jig for missing screws<br>- Check if fasteners are tight | - If screws are missing and cannot be replaced<br>- If fasteners continue to loosen |
| Jig Instability | - Jig shifts during use<br>- Increased rework/quality issues | - Verify jig is seated correctly<br>- Check for any obvious movement during operation | - If jig continues to shift during use<br>- If quality issues persist after checking stability |

## Work Orders & Parts Reference

| Issue | Related Work Order #s | Parts Often Needed |
| :--- | :--- | :--- |
| Loose or Missing Hardware | 15110 | 5/16-18 cap head screws, Loctite |
| Jig Instability | 19004 (Specialist repair) | None operator-replaceable |
