# Operator Troubleshooting Checklist: Tenoner A 517 Single End

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| Poor Cut Quality / Chipped / Dull Cutters / Uneven Gaps | - Parts are not gapped evenly<br>- Bottom bevel does not match top bevel<br>- Faint burned mark on product step<br>- Machine is cutting loudly<br>- Parts are coming back chipped from one side | - Visually inspect cutters (Station 1, Station 2) for chips or dull edges (machine OFF, LOTO)<br>- Observe how the material passes through the machine for any resistance or unusual pushing<br>- If cutting sounds are unusually loud or parts are chipped, stop the machine | - If cutters are visibly chipped, dull, or a full cutter is missing<br>- If cut quality issues (uneven gaps, burn marks, chips) persist<br>- If cutting sounds are unusually loud |
| Feed Track Grease Leaks / Marks on Product | - Black grease marks on top of the feed track<br>- Black grease on feed belt pads<br>- Grease marks on components passing through the machine<br>- Feed track full of oil | - Visually inspect the feed track and feed belt pads for grease accumulation<br>- Safely wipe visible excess grease from accessible areas (e.g., top of pads) with a shop rag | - If grease marks persist on the product after cleaning<br>- If there are visible leaks from fittings or hoses on the feed track system<br>- If the feed track is excessively oily underneath or continuously accumulates grease |
| Hold Down & Track Wheels Issues (Stuck, Loose, Damaging Parts) | - Hold down wheels are locking up or not rotating freely<br>- Wheels are causing drag or removing stickers from parts<br>- Guiding wheel on a station is broken<br>- Loose wheels in the infeed track | - With machine OFF and LOTO, visually inspect all hold down and track wheels for debris or damage<br>- Gently try to rotate each wheel by hand to check for free movement<br>- Carefully remove any visible debris that might be obstructing wheel movement | - If wheels are visibly damaged, have flat spots, are consistently stuck, or are excessively loose<br>- If parts are being marked, damaged, or labels are removed due to wheel issues<br>- If a guiding wheel is broken or missing |
| Machine Stopping / Alarms / Program Issues | - Machine stops and will not reset<br>- Machine stops with parts inside<br>- Alarms appear (e.g., "open file for station 4 tool data", "error during backslash edge guide") | - Ensure parts are loaded correctly, not too close together<br>- Verify the correct recipe/program is selected for the product being run<br>- Check for foreign objects or misaligned parts causing a jam<br>- Attempt a standard machine reset if available and safe (refer to operator manual) | - If the machine continues to stop without a clear cause after checking loading and recipe<br>- If error messages or alarms appear that you do not recognize or cannot clear<br>- If the machine will not reset |
| Infeed Guide / Table Alignment Issues | - Infeed guide is flexing or not properly aligned<br>- Side table is leaning or not level<br>- Infeed table is not even with other tables | - Visually inspect the infeed guide for any noticeable flexing or crookedness<br>- Check that all table sections are level and properly aligned with each other<br>- Remove any temporary clamps or shims (e.g., phenolic pieces) that might be causing misalignment | - If the infeed guide is flexing, crooked, or misaligned<br>- If any machine tables are not level or have shifted |
| Buffer Issues (Poor Finish / Crashing) | - Poor product finish (e.g., rounded front corners) after changing buffers<br>- Buffer is crashing into the part causing the machine to stop | - Double-check that the correct recipe/program is loaded for the part thickness<br>- If safe, visually observe the buffer's position relative to the part as it approaches or enters the station | - If product finish is consistently poor after buffer adjustments or changes<br>- If the buffer is physically crashing into parts, causing machine stops or damage |
| Dust Collection / Vacuum System Issues | - Vacuum black hose is broken or torn<br>- Dust collection bag has torn | - Visually inspect the vacuum hose for breaks, tears, or blockages<br>- Check the dust collection bag for any tears or holes<br>- If the dust bag is torn, safely replace it with a new one from spares (if trained and authorized) | - If the main vacuum hose is broken, torn, or severely damaged<br>- If dust collection performance is poor despite the bag being intact<br>- If you are unsure how to safely replace the dust bag |
| Track Feed Panel Issues | - A track feed panel or track pad has come off<br>- Machine has stopped due to a safety reason related to the track | - IMMEDIATELY stop the machine if a panel or pad comes off<br>- Do NOT attempt to operate the machine with missing or detached track components | - ALWAYS call maintenance immediately if a track feed panel or pad comes off, as this is a safety hazard and can cause further damage |
| Hold Down Movement Issues | - Machine is not moving when using the hold down button<br>- Servo appears to move, but the rest of the hold down mechanism does not | - Ensure the machine is ON and not in an emergency stop condition<br>- Listen carefully for any motor/servo sounds when pressing the hold down button<br>- Observe if any part of the hold down mechanism is attempting to move | - If the hold down mechanism does not move despite the button being pressed and the servo appearing to activate<br>- If the hold down feels loose or disconnected from its drive system |
| Tool Holder Removal Difficulty | - Difficulty removing a tool holder from its station | - Ensure the machine is fully OFF and locked out (LOTO) before attempting removal<br>- Visually inspect the tool holder and collet area for any debris or buildup that might be causing it to stick<br>- Gently try to remove the tool holder by hand | - If the tool holder remains stuck or requires excessive force to remove |
| Return System Belt Breakage | - One of the belts on the Tenoner return system has broken apart<br>- The return system is not functioning | - IMMEDIATELY stop the machine<br>- Visually inspect the return system belts for any breaks or damage | - ALWAYS call maintenance immediately if a drive belt in the return system breaks |
| Main Power Switch Detached | - The main power switch on the tenoner has come off completely | - DO NOT attempt to operate or reattach the switch<br>- Ensure the area is clear and safe | - IMMEDIATELY call maintenance if the main power switch becomes detached or damaged |
| Fence Edge Guide Issues | - Edge guide is not moving<br>- Back stop/edge guide is skewed affecting product finish | - With machine OFF and LOTO, visually inspect the guide for debris that may be obstructing movement | - If the guide remains stuck after clearing debris<br>- If the guide or back stop is skewed and requires realignment |
| Buffer Holder Hardware Failure | - Screw broke while removing buffer holder<br>- Unable to remove buffer holder due to broken screw | - STOP attempting to remove the screw to avoid further damage | - If a screw breaks or is stripped and cannot be removed by the operator |
