# Operator Troubleshooting Checklist: Edgebander Pop up table

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Lift Failure | - Lift is not working<br>- Table will not move | - Check the extension cord plug for damage or broken ground pins | - If the extension cord is damaged<br>- If the reel needs replacement |
| Table Leveling | - Table not lifting up evenly<br>- Leaking cylinder detected | - Inspect cylinders for visible leaks | - If cylinder is leaking<br>- If table remains uneven |
| Wheel Obstruction | - Wheel hitting fence<br>- Wheel not spinning<br>- Material being scuffed | - Check for clearance between the wheel and the wall/fence | - If the wheel needs to be shifted or repositioned |
