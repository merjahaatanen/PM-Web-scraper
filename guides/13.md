# Operator Troubleshooting Checklist: Notching machine, 1540 Door

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Drill Malfunction | - Drills will not activate<br>- Drill not going in (Right side)<br>- Motor bogging down or sounds abnormal<br>- Drill requires "jump starting" with bypass jogs | - Check if air lube reservoir is empty<br>- Check for visible broken wires on the right side<br>- Ensure air regulator is set above 85 psi | - If wires are cut/exposed<br>- If pneumatic valves (Solenoid/Pilot) fail<br>- If proximity sensor is defective<br>- If motor requires replacement |
| Cutting & Quality Issues | - Chipping product<br>- Scratches on doors<br>- Cutter cuts product before activation<br>- Notches not deep enough | - Verify door is inserted straight then moved left before clamping<br>- Check for worn cutter blades<br>- Check if drill bit has shifted relative to the guide | - To replace cutters or drill bits<br>- To adjust drill bit position (1/16" adjustments)<br>- To adjust cutter height/thread depth |
| Machine Stability | - Machine shaking during operation<br>- Parts/stoppers falling off | - Check if all machine legs are firmly supporting the frame<br>- Inspect clamps for loose stoppers | - If the motor belt is stretched or slit (Part # 3X473P)<br>- To replace missing stoppers |
