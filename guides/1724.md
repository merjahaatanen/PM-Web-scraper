# Operator Troubleshooting Checklist: O-Sama (Joos) Glue Spreader

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| Glue Leaks & Valve/Hose Issues | - Leaking glue from rollers<br>- Leaking glue from side plates<br>- Leaking glue from bottom rollers<br>- Leaking glue from bottom pan<br>- Leaking glue from a hose line<br>- Knob or handle broke on a valve<br>- Glue hose is ripped<br>- Clamp holding a glue hose is broken | - Check for and remove any accumulated dry glue from roller sides and corners.<br>- Ensure side plates are properly positioned against the rollers.<br>- Apply grease to the sides of the rollers daily as part of pre-setup.<br>- Check glue hoses for visible damage or disconnections.<br>- Check hose clamps for tightness and damage. If easily replaceable, replace them. | - If glue leak persists after cleaning and greasing.<br>- If a valve knob or handle is broken.<br>- If a glue hose is ripped or cannot be patched by an operator.<br>- If a hose clamp cannot be replaced by an operator.<br>- If the water catch tray has a hole or crack.<br>- If a shut-off valve is broken and leaking. |
| Roller Operation & Spreading Problems / Noise | - Machine making unusual noises (e.g., grinding, slipping chain)<br>- Glue line rollers show excessive wear (no keyhole/set screws)<br>- Top roller is not spreading glue evenly<br>- Rollers are at the end of their lifespan | - Use the laminate strip to set the proper distance from the glue rollers and glue trays daily.<br>- Clean the end of the glue tray every day.<br>- Ensure the handle separator on the glue spreader is tight. | - If machine is making grinding or unusual noises that persist.<br>- If glue rollers are not spreading glue evenly.<br>- If glue rollers appear worn out or damaged (e.g., no keyhole, loose set screws).<br>- If the drive chain is slipping or making loud noise.<br>- If the handle separator is loose and you cannot tighten it safely. |
| Air Table & Material Handling Issues | - Infeed table belt staples showing excessive wear<br>- Water tray rollers are not free-rolling<br>- Air table sometimes won't work<br>- Air table flap is broken<br>- Particle board air cleaner is not activating<br>- Air cleaner for dust particles is not working<br>- Air table is chipping edge laminate on pieces | - Check for any loose screws on the air table that might be high and hitting material.<br>- Inspect the air table flap for obstructions or damage.<br>- Ensure the activation switch for the air cleaner is not shifting or stuck. | - If infeed table belt shows excessive wear.<br>- If water tray rollers are jammed or not turning freely.<br>- If the air table is intermittent or not working at all.<br>- If the air table flap is broken or missing.<br>- If the particle board air cleaner is not activating.<br>- If the air table is chipping the product laminate. |
| Rollers Not Rolling / Interlock & Electrical Problems | - O-Sama rollers stopped rolling<br>- Rollers not activating or able to pass pieces<br>- Machine not activating<br>- Rollers not spinning even after checking e-stops<br>- A part is preventing the glue roller from spinning | - Ensure all safety doors and covers are securely closed and latched.<br>- Check that no E-stop buttons are pushed in.<br>- Look for any visible loose hardware on safety interlock switches on gates/covers. | - If rollers are still not moving after checking safety doors, covers, and E-stops.<br>- If a safety switch or interlock seems faulty or loose.<br>- If the machine has no power or is not responding to controls. |
| Misting System Issues | - Misting system connector broken<br>- Misting system not operating<br>- Glue drying up more than usual | - Check for any visible breaks or disconnections in the misting system's hose line. | - If the misting system is not working or has broken components that cannot be easily fixed. |
| Glue Tote Replacement | - Glue tote needs to be replaced | - Carefully replace the glue tote.<br>- When handling the glue nozzle, be very gentle as plastic parts can break easily. | - If you are unable to safely replace the glue tote.<br>- If the glue nozzle breaks during replacement. |
