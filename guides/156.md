# Operator Troubleshooting Checklist: Return System, Thomas, 1/2" Solid Panels

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Belt Failure | - Broken or snapped orange belt<br>- Ripped long transfer belt<br>- Broken infeed belt<br>- Stretched belts causing parts to catch | - Inspect belts for rips, snaps, or stretching | - Belt needs full replacement<br>- Belt requires melting, gluing, or cutting/shortening |
| System Alignment | - Return system has shifted<br>- Section of system sitting too high<br>- Potential for panel damage | - Visually check if the table or roller assembly is level | - Shaft set screws need loosening<br>- Roller assembly or table needs adjustment |
| Bearing Noise | - Grinding noise coming from return system<br>- Uneven rotation | - Listen for abnormal noises at bearing locations | - Bearing is broken or grinding into shaft<br>- Bearing requires replacement |
