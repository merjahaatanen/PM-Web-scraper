# Operator Troubleshooting Checklist: Chop saw by Holzma Saw

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Blade Wear | - Blade is dull<br>- Cutting phenolic causes smoking<br>- Poor cut quality | - Check for visible dullness<br>- Replace blade with a sharpened one | - If blade cannot be replaced by operator |
| Vacuum System Failure | - Vacuum stuck ON overnight<br>- Vacuum not triggering during use | - Check if the vacuum is responding to saw operation | - If the limit switch or torsion spring is broken |
| Power/System Failure | - Machine stopped working<br>- Machine "not working any more" | - Check screen for "Low Air Pressure" error<br>- Verify main compressor is running<br>- Ensure you are not cutting multiple thick pieces (e.g., 2x 8" phenolic) | - If breaker L41-10 is tripped<br>- If air pressure remains low after compressor check |
| Mechanical Jam | - Blade will not return to home position<br>- Pieces not cut all the way through | - Reset the blade carrier all the way down<br>- Power cycle the machine (Turn OFF and ON again)<br>- Adjust the speed control knob<br>- Press "Return" and "Start" buttons again | - If the Upper Guide Beam Cylinder remains stuck |
