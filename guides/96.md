# Operator Troubleshooting Checklist: Vacuum Lift 1080 Line packout

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Suction Cup Alignment | - Cup leaning to the side<br>- Cup is not level | - Visually inspect the pin holding the cup level | - If the pin is worn out or wedged<br>- If the pin needs replacement |
| Unusual Noise | - Popping noise during operation | - Check if sheets are sticking to each other<br>- Separate stuck sheets | - If noise persists after clearing material |
| Movement Issues | - Vacuum lift gets stuck<br>- Must force lift to move | - Check for obstructions around the lift area | - If lift continues to stick without obvious cause |
