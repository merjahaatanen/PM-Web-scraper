# Operator Troubleshooting Checklist: Stacker R-19

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Wheels & Tires | - Banging noise during travel<br>- Unit not tracking straight on ramps<br>- Uneven balance between left/right sides<br>- Visible wear/flat spots on wheels | - Inspect wheels for flat spots<br>- Check for debris or wood on travel path | - Any wheel needs replacement<br>- Drive tire is spinning/not gripping<br>- Steering tire shows heavy wear |
| Brake & Startup | - Error code 48 or 84 at startup<br>- Unit spins 360 degrees while driving<br>- Fault occurs specifically in the morning | - Try turning unit OFF<br>- Manually push the unit<br>- Turn unit back ON to see if fault clears | - Error codes persist after restart<br>- Brake assembly failure suspected<br>- Master cylinder needs replacement |
| Electrical & Safety | - Horn not working<br>- Blue backup light not facing rear | - Test horn button for physical failure<br>- Check if light bracket is loose or broken | - Horn button is faulty (requires handle re-teaching)<br>- Backup light bracket is broken |
| Hydraulic Leaks | - Hydraulic fluid pooling on floor | - Check around the base for visible leaks | - Leak is internal to the body frame<br>- Fluid continues to leak after inspection |
