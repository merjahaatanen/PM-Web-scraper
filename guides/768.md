# Operator Troubleshooting Checklist: Vacuum Lift (Anderson CNC)

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Vacuum Leaks | - Losing pressure when picking up components<br>- Unable to lift doors<br>- Not lifting components properly | - Inspect all hoses for visible holes or blow holes<br>- Check for air leaks along the main and section hoses | - Leak cannot be sealed<br>- Hose requires full replacement |
| Filter & Motor Maintenance | - Vacuum not working<br>- System seems clogged<br>- Rattling noise from system | - Check vacuum filter for dust buildup<br>- Inspect vacuum pan and motor for heavy dust accumulation | - Filter is too clogged to clean<br>- Motor belt requires conditioner<br>- Cover removal required for cleaning |
| Pad & Hardware Wear | - Losing suction on bigger pieces<br>- Handle came apart<br>- Lift fails to hold components | - Inspect suction pads for warping<br>- Check if pads are sealing properly on the part | - Pads are warped and need replacement<br>- Handle or mechanical parts are broken |
| Mechanical Failure | - Broken spring on lift | - Visually inspect lift springs and pins | - Spring or pin is broken and needs replacement |
