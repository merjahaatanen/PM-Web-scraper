# Operator Troubleshooting Checklist: Dust Collector, Donaldson Downflo Oval

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Filter Performance | - Controller numbers at lowest point<br>- Numbers above normal<br>- Dust blowing out of exhaust during cleaning cycle<br>- Excessive blow-by outside | - Check controller numbers for high/low readings<br>- Visually inspect exhaust for dust leakage | - When filters need replacement<br>- If filters are ripped or worn out |
| Belt & Motor Noise | - Screeching noise from motor/pulleys<br>- Loud non-stop screeching | - Identify if noise is coming from the belt area | - When belt tension needs adjustment<br>- If belts have stretched due to wear |
| Hopper & Rotary Valve | - Rotary valve not turning<br>- Hopper not working<br>- Hopper backed up | - Check if the rotary valve is rotating<br>- Check for visible debris stuck in the valve | - If fuses are blown (KTK-R-50)<br>- If hopper motor needs replacement<br>- If material is wedged in the valve |
| System Power | - Machine stops running during production | - Check if the machine is powered on | - If thermal overload has tripped<br>- If fuses are blown |
| Vent & Body Damage | - Vent is caved in<br>- Loud metal banging noise | - Inspect vents for punctures or caved-in sections<br>- Listen for loose rods or shaking parts | - To replace membrane/covers<br>- To fix broken dampener lever rods |
| Flame Ex System | - Fire alarm activated<br>- Water spraying in duct | - Check drilling machines for dull bits (smoke, flare-ups, or blown-out holes)<br>- Silence alarm using code 3333 via guide on panel | - If water valve is stuck open<br>- If solenoid diaphragm is dirty<br>- If infrared sensors need cleaning |
