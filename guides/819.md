# Operator Troubleshooting Checklist: Scissor Lift Holzma Saw

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Lift Table Power/Movement | - Lift does not go up or down<br>- No power to controller<br>- Lift only moves in one direction | - Check that the power cord is fully plugged in<br>- Inspect the foot switch for loose cable connections | - If power cord is damaged/cut<br>- If wires inside the harness are broken<br>- If internal electrical connections are loose |
| Infeed Issues | - Difficulty feeding sheets<br>- Infeed arm is stuck | - Use the blow nozzle to clear debris from the area<br>- Clean the infeed arm area to prevent sticking<br>- Ensure the blow nozzle is pointed in the correct direction | - If the arm remains stuck after cleaning |
| Blade Cover Issues | - Blade cover came loose<br>- Door not moving correctly | - Check if the cover is physically loose | - If the pivot point screw is wrong or needs replacement |
