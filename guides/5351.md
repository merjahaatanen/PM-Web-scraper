# Operator Troubleshooting Checklist: VLM Storage Lift -Small Hardware

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Software & Connectivity | - Computer and VLM not connecting<br>- Power Pick software not opening<br>- Protocol Initialization Failed<br>- Random loss of connection | - Verify computer is connected to "BobNet" wifi<br>- Shut down the computer and VLM machine properly<br>- Restart the VLM machine<br>- Restart the associated computer | - If rebooting does not restore connection<br>- If wifi signal remains weak<br>- If adapter settings need updating |
| Sensor & Tray Errors | - Error 1915: Tray in opening position<br>- Error 1979: Vertical photo cell defective<br>- Error 2101: Safety Circuit<br>- Error 2203: Check sensor(s)<br>- Error 1926: Height control interrupted | - Manually push bottom tray back if it is too far forward<br>- Manually send trays back and restart order<br>- Check for fallen hardware or obstructions near bottom lasers using a flashlight | - If sensors are blinking red/white<br>- If the light barrier housing is misaligned<br>- If a limit switch is in the wrong position<br>- If firmware updates are required |
| Mechanical & Jamming | - Trays getting stuck<br>- Package slip dispenser jamming<br>- Extraction not aligned with trays | - Clean the package slip dispenser pathway and remove excess tape<br>- Wipe down dust weekly to prevent tray sticking | - If trays require manual pulling to clear errors<br>- If the machine requires recalibration due to belt stretch |
| Display Issues | - VLM display bar not showing information<br>- Beam display not working | - Turn both the VLM machine and the computer OFF and then back ON to reset | - If the display remains blank after a full system reset |
