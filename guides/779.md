# Operator Troubleshooting Checklist: Router Station (Stile Building cell)

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Cutting Quality | - Blade/Bit is dull<br>- Poor cut quality on steel cores | - Inspect router bit for wear<br>- Replace the router bit (Standard Operator Function) | - If bit replacement does not resolve the issue |
| Vacuum System | - Vacuum lift hose is ripped<br>- Air leaking from hose | - Inspect blue hose for holes/tears<br>- Apply duct tape to seal small leaks | - If leak is too large for tape to seal<br>- If hose requires full replacement |
