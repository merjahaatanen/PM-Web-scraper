# Operator Troubleshooting Checklist: POP UP Table, 3/4" Edge Finisher

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Air System Leaks | - Audible air leak from air line<br>- Air regulator leaking air | - Check regulator adjustment knob<br>- Ensure regulator is not turned up to max; lower it to see if leak stops | - Leak continues after adjusting regulator<br>- Air hose is damaged or punctured<br>- Regulator requires full replacement |
| Kick Plate / Hinge Failure | - Kick plate hinge broke on left side<br>- Hinge loose or broken<br>- Table unable to stay up/down | - Inspect hinge for visible breaks or loose screws | - Hinge needs replacement<br>- Bumper needs to be installed to prevent future breaks |
| Power Loss | - Table is not working / No power | - Check for visible damage to the main power cord | - Electric cord is smashed or wires are disconnected |

## Work Orders & Parts Reference

| Issue | Related Work Order #s | Parts Often Needed |
| :--- | :--- | :--- |
| Air System Leaks | 7070, 7421, 13083 | None operator-replaceable |
| Kick Plate / Hinge Failure | 3199, 3201 | None operator-replaceable |
| Power Loss | 4795 (Specialist repair) | None operator-replaceable |
