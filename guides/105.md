# Operator Troubleshooting Checklist: Drilling Machine CNC, 1040 Laminate

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Vacuum Table Seal | - Vacuum seal broken<br>- Gasket worn out | - Inspect rubber seal for rips or wear | - Seal requires replacement |
| Power & Boot Issues | - Machine doesn't load program on startup<br>- Screen shut off/won't turn on<br>- No power to computer | - Shut machine down completely<br>- Wait one minute<br>- Restart computer and reload program | - Computer power supply failure<br>- Beckhoff drive error codes (0.3, N.17) |
| Sensor & Positioning | - Sensors not picking up to drill<br>- Drilling locations off by +/- 1/8"<br>- Sensors not activating | - Ensure parts are aligned with guides<br>- Clean sensor face with alcohol<br>- Check if tape is being used to trigger sensors | - Sensors are physically broken<br>- Offset adjustments needed for Stop 1 and Stop 4 |
| Software & Programming | - File not found error<br>- Low disc space on C drive<br>- Program glitch (incorrect drill locations) | - Restart computer<br>- Manually select drilling program<br>- Delete old jobs from C drive to free space | - Corrupted program files<br>- Need to restore backup files to correct location |
| Mechanical & Air | - Air leak from Z axis cylinder<br>- Drill spinning but not descending<br>- Drilling speed too slow | - Verify correct drill bit is installed for the job<br>- Check air lines for visible rips | - Air hose replacement needed<br>- Stuck solenoid valve requires lubrication |
