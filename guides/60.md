# Operator Troubleshooting Checklist: Laminate Slitter

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Cutter & Material Buildup | - Loud screeching noise<br>- Motor stops midway through cut<br>- Cutters loaded with scrap/fall off | - Stop machine<br>- Disassemble cutters to remove scrap material<br>- Clear trash from underneath the machine<br>- Ensure blade extractors are loose when cleaning | - If cutters need sharpening or replacement<br>- If blade extractors are wedged/stuck in metal |
| Power & Connection | - Slitter won't start<br>- Power cable feels loose<br>- No connection at the plug | - Check if power cable is disconnected<br>- Ensure machine is unplugged from wall socket (not short plug)<br>- Tighten hardware on connections | - If the machine still won't power on<br>- If a fuse needs replacement |
| Drive & Rotation Issues | - Slitter is not rotating<br>- Motor spinning but cutters not moving<br>- Chain slipped off | - Check for two plastic sheets fed at once (causing jams)<br>- Clear any plastic stuck in the mechanism | - If the drive chain or sprocket is out of alignment<br>- If the gearbox is leaking fluid<br>- If the motor key/pin has rolled over |
| Component Alignment | - Not functioning/impacting production<br>- Cutting wheels off alignment | - Inspect cutting wheels for visible misalignment | - If wheels or sprockets require mechanical adjustment |
