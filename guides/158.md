# Operator Troubleshooting Checklist: Return System, Thomas, Laminate Trimmer

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Orange Belt Failure | - Belt ripped or snapped<br>- Middle section not moving<br>- Belt is loose<br>- Parts getting stuck | - Check if belt is fully detached or just loose<br>- Inspect for warped panels or excessive plastic overhang causing the rip | - Belt requires replacement<br>- Belt needs professional gluing/tensioning<br>- Belt is snapped |
| Surface Damage/Scratching | - Scratches on bottom of HPL<br>- Material dragging<br>- Metal plate sticking up | - Inspect return system for debris under transition plates<br>- Check if all rollers are spinning freely | - Roller needs replacement<br>- Transition metal plate is bent or too high<br>- Rubber band missing from pulley |
| Control & Power Issues | - Speed control not working<br>- Controller "ON" light is off | - Verify power source is connected | - Speed potentiometer needs soldering<br>- Controller bulb needs replacement |
| Mechanical Alignment | - Outfeed shaft not sitting correctly | - Observe if shaft has shifted during operation | - Shaft needs to be repositioned and locked in bearing |
| E-Stop Failure | - E-stop button came off<br>- E-stop button getting stuck | - Check if button is physically detached | - E-stop requires replacement |
