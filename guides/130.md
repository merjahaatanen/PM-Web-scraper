# Operator Troubleshooting Checklist: Vacuum Lift 1/2" panels packout

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Suction Loss | - Panel slips off suction cups<br>- Lost suction and dropped panel | - Test suction with a piece of panel board<br>- Check if vacuum is working normally during a test run | - Suction continues to fail<br>- Hose is leaking or losing pressure |
| Suction Cup Failure | - Suction cup leaning down<br>- Spring on suction cup is broken | - Inspect suction cups for visible damage or leaning | - Pin is worn out<br>- Spring needs replacement |
| Handle & Trigger Issues | - Trigger getting stuck<br>- Vacuum not working correctly<br>- Handle screws missing | - Clean the handle release enclosure<br>- Check for missing screws on the handle | - Trigger remains stuck after cleaning<br>- Screws cannot be tightened/replaced |
| Handle Positioning | - Handle gets stuck underneath side table<br>- Handle hangs too low when not in use | - Ensure handle is hanging freely | - Handle needs to be raised/repositioned |
