# Operator Troubleshooting Checklist: Return System, Thomas, 1040 Edgebander

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| Belts Loose, Broken, Worn, or Off-Track | - Return system belts appear too loose<br>- Belts are making noise<br>- A small band looks like it's about to snap or break<br>- A belt looks ripped, flattened, or is barely holding on<br>- A band has broken or snapped off<br>- A belt has popped off or is out of its track<br>- The back end of the return system is not moving<br>- A band is causing components to fall off<br>- Rubber bands are missing | - Inspect all belts for visible damage, rips, or if they are off track<br>- If a belt is visibly off track and accessible, try to gently guide it back onto its track<br>- Check for any belts that appear stretched out | - If belts are loose and you cannot safely adjust them<br>- If any belt is damaged, ripped, broken, or appears stretched<br>- If a belt keeps popping off its track<br>- If the back end of the return system is not moving and you can't identify a simple fix<br>- If a band is causing parts to fall off<br>- If any rubber bands are missing |
| Parts Getting Stuck or Not Returning (General) | - Components are getting stuck at the back end<br>- Pieces keep getting stuck at the end<br>- Parts are not returning and getting stuck at the end/entrance<br>- Orange belts appear to be struggling to move panels<br>- Components are twisting as they come out | - Check for any obvious obstructions on the return path<br>- Observe if any V-belts look twisted<br>- Gently clean drive belts that appear covered in debris<br>- Note if large or unusual panels were recently run, as this can affect alignment | - If parts are consistently getting stuck or not returning<br>- If orange belts are struggling to move panels<br>- If components are twisting<br>- If you notice a V-belt twisted or any other misalignment<br>- If cleaning belts doesn't resolve the issue |
| Return System Rollers or Pulleys Malfunctioning | - The return system at the very end is not working<br>- The mid-section of the return system is not spinning or barely spinning<br>- You suspect rollers need to be replaced | - Check if any wheels or pulleys are seized or stiff<br>- If approved by Maintenance, apply WD40 to visible wheels/pulleys to free them up | - If applying lubricant does not resolve the issue<br>- If wheels or pulleys appear damaged, broken, or are not spinning freely<br>- If the mid-section is barely spinning |
| Panels Chipping, Damaged, or Incorrect Height/Alignment | - Panels are returning with chips on their edges<br>- Panels are getting stuck or turning sideways at the end<br>- Boards are hitting and getting stuck because the return system seems too high<br>- Warped panels seem to be exacerbating issues | - If approved by Maintenance, spray the bottom of the belts with silicone spray to aid panel movement<br>- Observe if warped panels are consistently causing the issue | - If panels are chipping or getting damaged<br>- If panels are getting stuck or turning sideways<br>- If the return system height appears incorrect (too high or misaligned)<br>- If silicone spray doesn't help with panel movement |
| Return System Not Working (Electrical or Controls) | - Return system is not working properly, or stops intermittently<br>- The system requires manual assistance to move parts | - Check if any control knobs, such as a potentiometer knob, feel loose or easily move out of position | - If the return system is not working at all or stopping intermittently<br>- If you find a loose control knob or suspect an electrical issue<br>- If production requires manual assistance to move parts |
| Missing Rollers | - Bar rollers are missing before the component returns to the operator | - Check the immediate area around the machine for any rollers that may have been intentionally removed and placed aside | - If any bar rollers are missing and cannot be located nearby, or if they are found and need re-installing |
