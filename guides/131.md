# Operator Troubleshooting Checklist: Vacuum Lift Laminate Pack Out

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Vacuum Lift Failure | - Vacuum does not lift up<br>- Unit lowers by itself | - Inspect the blue vacuum hose for rips or tears<br>- Check vacuum filter for clogs | - If hose is ripped and needs replacement<br>- If filter replacement does not fix the lift |
| Suction Cup Failure | - Broken suction cup spring | - Inspect suction cup spreader bar for broken springs | - If a spring is broken and needs replacement |

## Work Orders & Parts Reference

| Issue | Related Work Order #s | Parts Often Needed |
| :--- | :--- | :--- |
| Vacuum Lift Failure | 5923 | 20 FT blue vacuum hose, vacuum filter |
| Suction Cup Failure | 11629 | Suction cup spreader bar spring (#5637) |
