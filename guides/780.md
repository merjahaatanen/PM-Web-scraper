# Operator Troubleshooting Checklist: Drill Press (Stile Building Cell)

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Drill Alignment | - Drill is out of line<br>- Drilling is off center<br>- Leveling bar sticking out | - Verify if the issue is specific to 3'' stiles (known alignment variance) | - If drill head is not straight<br>- If alignment requires adjustment |
| Press Clamp Hardware | - Missing bolts<br>- Loose clamps | - Visually inspect clamp bolts for missing nuts | - If bolts are loose or nuts are missing<br>- To apply Loctite or replace lock nuts |
