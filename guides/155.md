# Operator Troubleshooting Checklist: Shaper, Single Spindle, Northfield

## Troubleshooting
| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
| :--- | :--- | :--- | :--- |
| Blade Performance | - Raggedy finish/visible wood grains<br>- Blade feels too tight<br>- Material requires clamps to close during assembly | - Inspect blade for dullness or damage<br>- Verify blade kerf is correct (.135)<br>- Replace blade with a new, smooth-finish blade | - If blade replacement does not resolve the finish quality<br>- If the blade is warped |
| Cutting Alignment | - Stile slit not centered<br>- Center cut line shifted<br>- Stiles lifting up during cut | - Check if the blade is dull (common cause of off-center cuts)<br>- Replace the blade<br>- Verify handwheel has not been accidentally moved | - If the height adjustment needs resetting to 3/8 cut line<br>- If the center line has shifted and requires mechanical calibration |
| Machine Power/Noise | - Loud noise from cutter<br>- Machine trips and will not turn back on | - Power down machine immediately | - Always call maintenance for loud noises, tripped breakers, or warped blades causing heat spots |
