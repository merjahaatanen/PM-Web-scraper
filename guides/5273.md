# Operator Troubleshooting Checklist: Edgebander Homag 2520 Servo 6 Coil

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| **Edging & Material Feeding Problems** | - Machine taking multiple edgings at a time<br>- Edging not being fed through the machine<br>- Edging getting stuck on the edge clamp pop up<br>- Edging material is curved and causing issues<br>- Edging coming out too short<br>- Machine shuts down, unable to clear the track<br>- Banding on the return system came off track<br>- Insufficient glue applied (e.g., only top/bottom) | - Check that the edging material is straight and not curved<br>- Check for blockages in the material feed path and clear them<br>- Manually clear any steel core stiles or edging stuck in the machine track<br>- Check if the banding grip is slipping and try to re-seat it<br>- Ensure the glue pot is clean and free of old, stuck glue | - If you find a broken piece of metal in the edging feed path<br>- If the air regulator or spring metal needs adjustment for thicker material<br>- If a suction cup is ripped or its holder is broken<br>- If the banding grip continues to slip after re-seating<br>- If the return system rubber cord is loose, damaged, or a wheel is cracked<br>- If cleaning the glue pot does not resolve uneven glue application |
| **Emergency Stop & Machine Start-up Errors** | - Machine will not run after turning it on<br>- Error message "Edge 1/ Control computer /Profile trimming 1 left. Unit front edge at the top not in parking position"<br>- Machine displays "Local emergency stop chain interrupted"<br>- Machine displays "e-stop chain error"<br>- General machine error code at startup | - Ensure all E-stop buttons are disengaged (pulled out)<br>- Check that all safety doors and guards are securely closed<br>- Inspect the return system for any rubber cables popped off track; if found, clear the track and reset the cable (if easily accessible)<br>- If an error refers to a specific unit (e.g., trimming unit), visually inspect it for anything obviously out of place | - If the machine still won't run after checking E-stops and doors<br>- If the error message is specific to a unit (e.g., trimming unit) and you can't see an obvious fix<br>- If resetting E-stop buttons doesn't clear the "emergency stop chain interrupted" error<br>- If you suspect an E-stop module is loose or faulty |
| **General Mechanical & Component Damage** | - A grease fitting is broken with part still inside<br>- Side rail on machine has damaged rollers<br>- Infeed table does not stay locked in place<br>- Switch to activate the strips holder fell off<br>-air cylinder test test | - If a switch has fallen off, carefully reattach it if it's a simple snap-in component and safe to do so | - For any broken grease fittings<br>- For damaged side rail rollers<br>- If the infeed table drop pin or bushing is misaligned and not locking<br>- If a switch is broken or cannot be easily and safely reattached |
| **Glue System Issues** | - Glue taking a long time to heat up<br>- Machine errors related to glue pot after startup<br>- General glue pot maintenance required | - Check the glue reservoir level and manually refill if it's low<br>- Ensure the glue reservoir is turned ON<br>- Regularly remove old, stuck glue from the glue pot<br>- Clean the glue level probe | - If the glue still doesn't heat up properly after checking level and power<br>- If cleaning the glue pot doesn't resolve issues<br>- If glue pot maintenance requires more than basic cleaning |
| **Rough Edges & Cut Quality** | - Machine cutters leaving rough edges on top and bottom<br>- Top and bottom edges are rough after processing | - Check the dust hoses for the cutters for any blockages and clear them<br>- Report if blades appear dull or worn | - If rough edges persist after checking dust hoses<br>- If cutters need adjustment or replacement<br>- If any part of the cutter assembly (e.g., a bracket) is broken |
| **Power & Intermittent Shut-Off** | - Machine is randomly shutting off during production<br>- Lost power to the Edgebander, tried to reset but it won't turn on | - Check the power cord and any extension cord connected to the machine to ensure they are securely plugged in and undamaged | - If the machine continues to randomly shut off<br>- If the machine has completely lost power and will not turn on after checking power connections |
| **Drive Belt Issues** | - Orange belts are loose<br>- Orange belts coming out of track | - Visually inspect belts for slack or misalignment | - If belts need to be tightened or replaced |
