# Operator Troubleshooting Checklist: Saw, 10' Panel, Laminate Line

## Troubleshooting

| Issue | Symptoms | Troubleshooting Steps | When to Call Maintenance |
|---|---|---|---|
| Saw Not Activating / Stuck / Unexpected Stop | - Saw does not move / turn on / activate cut<br>- Saw goes all the way up and gets stuck<br>- Saw stopped unexpectedly<br>- Saw requires multiple button pushes to activate | - Check if the E-stop button is fully disengaged and reset.<br>- Ensure the saw carriage is not past its normal travel limits.<br>- Visually inspect all accessible limit switches for obstructions, looseness, or if they appear stuck.<br>- Gently work the limit switch plungers by hand if they appear stuck.<br>- Check the start button to ensure it is not loose or visibly damaged.<br>- Inspect for loose or disconnected air hoses, especially near the blade mechanism. If disconnected, firmly reattach.<br>- If the saw goes all the way up, visually check if the limit switch arm is loose.<br>- Check for excessive grease or debris on the stop plate that might prevent proper limit switch activation, and clean if necessary.<br>- Confirm the travel speed setting is not too low. | - If saw still does not move after checking limit switches, start button, and air hoses.<br>- If any wires are visibly broken, frayed, or if fuses are blown.<br>- If air hoses are damaged or fittings are broken.<br>- If the motor is making loud noises or not spinning.<br>- If the counterweight has come off its track or has broken components.<br>- If a start button feels faulty or doesn't consistently engage.<br>- If a limit switch arm is loose and requires adjustment with tools. |
| Physical Damage / Misalignment / Loose Components | - Hydraulic pump or motor not secured<br>- Top hold-down piston not working correctly (blow-by)<br>- Hold-down shifted or looks bent<br>- Gap between backboard and metal ledge, material getting stuck<br>- Rear counterweight bracket broken or off track<br>- Rod on bottom of beam not closing or looks bent<br>- Heard a bang from the rear of the machine (and stopped it)<br>- Magnetic material holder not holding<br>- Bottom part of the fence getting stuck when closing<br>- Panel saw resting bars out of alignment<br>- Guide on panel saw loose or uneven | - Visually inspect the hydraulic pump and motor for loose mounting bolts.<br>- Check the fence for tightness and look for gaps where material might get stuck.<br>- Inspect the magnetic material holder for visible damage.<br>- Check the bottom fence holder for any obvious obstruction or damage if it's binding.<br>- Visually check the counterweight system for any obvious breaks or if it has come off track.<br>- Visually check resting bars and guides for misalignment or loose parts. | - If bolts are broken, missing, or require tools to tighten on major components.<br>- If hydraulic cylinders or pistons are leaking, bent, or not functioning correctly.<br>- If a counterweight bracket or guide bar is broken or worn out.<br>- If the magnetic material holder housing is broken.<br>- If welding is required for any broken components.<br>- If fence or guide adjustment cannot be resolved by simple tightening or repositioning by hand. |
| Cutting Not Straight / Crooked / Bad Finish | - Saw is not cutting square / straight / at an angle<br>- Laminate sheets are coming out cut crooked<br>- Material has a bad finish / debris buildup on blades<br>- Smell of burning while cutting<br>- Dust collection seems less effective when cutting plastic | - Check the blade condition for dullness or excessive debris buildup. Replace blade if necessary.<br>- Verify the bottom fence is level using a suitable tool (e.g., spirit level).<br>- Check if incoming laminate material is consistently square.<br>- Visually inspect for noticeable play or misalignment in the blade path or guides. | - If replacing the blade does not resolve dullness or burning smell.<br>- If cuts are still crooked or not square after checking the fence and incoming material.<br>- If motor alignment is suspected of being off.<br>- If leveling brackets are broken or require adjustment with tools. |
| Dust Collection Issues | - Dust collection not working properly<br>- Visibly ripped or damaged duct hoses | - Check the blade condition; a dull blade can produce more dust and reduce collection efficiency.<br>- Visually inspect dust collection hoses for rips or disconnections. If disconnected, firmly reattach. | - If hoses are ripped beyond simple reattachment.<br>- If dull blade replacement doesn't improve dust collection.<br>- If dust collection performance is poor without obvious hose damage or dull blade. |
| Hydraulic Leaks | - Hydraulic manual operated valve leaking from fitting<br>- Control valve leaking from rear handle rotation | - Visually inspect fittings and valves for signs of oil leaks. | - Immediately if any hydraulic fluid is leaking from fittings or valves. |
| Drag Fault | - Machine giving "drag fault" error frequently | - Visually inspect grippers (especially the 3rd one from left) to see if they are dragging on the table top.<br>- If a gripper is dragging, check if it can be easily repositioned by hand without tools. | - If the "drag fault" persists after checking grippers.<br>- If gripper adjustment is not possible or requires specialized tools. |
